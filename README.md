# 🏠 Casa Aberta Senac

Projeto desenvolvido para o evento **Casa Aberta do Senac**, com o objetivo de proporcionar uma experiência digital para os participantes, incluindo controle de acessos, feedbacks e interatividade.

---

## 📋 Descrição

Este sistema integra funcionalidades web para registro e controle de presenças, painéis em tempo real e feedback via QR Code. Foi desenvolvido com tecnologias modernas para backend e frontend, facilitando o gerenciamento do evento e melhorando a experiência dos participantes.

---

## 🚀 Funcionalidades

- ✅ Registro de presença via totem touchscreen  
- ✅ Controle de acesso e registro em banco de dados  
- 📊 Dashboard em tempo real para visualização de dados  
- ✉️ Sistema de feedback interativo via QR Code  
- 🎉 Painel de agradecimento e confirmação após envio do feedback  
- 📱 Interface responsiva com Bootstrap  

---

## 🛠️ Tecnologias utilizadas

- 🌐 HTML5, CSS3, JavaScript  
- 🎨 Bootstrap  
- ⚙️ Node.js (backend)  
- 🗄️ MySQL (banco de dados)  
- 🖥️ XAMPP/Apache (servidor local)  
- 📈 Power BI (para análises e dashboards)  
- 🔗 QR Code para interatividade
